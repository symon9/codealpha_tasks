## 📋 Task Overview

### ✅ Task 1 Wireframing & Low-Fidelity Design

Low-fidelity wireframes for a fintech mobile application covering five core screens: Login, Dashboard, Transfer, Schedule Transfer, and Success state. Designed in Figma following standard wireframing conventions.

🔗 **LinkedIn Post:** <https://www.linkedin.com/posts/simon-emmanuel_uiux-fintech-productdesign-activity-7440588839796187136-xXxF>

### ✅ Task 2 High-Fidelity UI Design

High-fidelity conversion of the Task 1 wireframes into a polished, production-ready UI. Includes a complete design system with color palette, typography, component library, and responsive layouts across five screens.

🔗 **LinkedIn Post:** <https://www.linkedin.com/posts/simon-emmanuel_uidesign-uxdesign-codealpha-activity-7444493293947731968-hXZv>

### ✅ Task 3 UX Case Study

A structured UX audit of the **Duolingo** language-learning app. Covers user persona definition, user journey mapping, UX strengths and weaknesses analysis, and three actionable improvement proposals with supporting mockups.

🔗 **LinkedIn Post:** <https://www.linkedin.com/posts/simon-emmanuel_uiux-uxdesign-casestudy-activity-7445032282391859200-yHUp>

### ✅ Task 4 Mini Project: Prototype Design

A complete high-fidelity clickable prototype for **Cineora** a Netflix-style mobile streaming app designed for the African market. Features 9 fully connected screens, NGN subscription pricing, and a dark cinematic design system. Built entirely in Figma.

🔗 **Prototype Link:** <https://bit.ly/4msOkVs>
